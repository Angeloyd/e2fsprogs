This is the new version (1.42.7) of the second extended file

system management programs.

From time to time, I release new versions of e2fsprogs, to fix

bugs and to make the utilities more robust. You can always find information about the latest version at the the e2fsprogs web page, which is:

http://e2fsprogs.sourceforge.net

The INSTALL file has instructions on building and installing

e2fsprogs. Provisions for building Red Hat RPMs and Debian dpkg files are supplied as well.

In case of bugs in these programs, please contact Ted Ts'o at

tytso@mit.edu or tytso@alum.mit.edu. See the e2fsck man page for suggestions of what sort of information to include when submitting bug reports for these programs.
